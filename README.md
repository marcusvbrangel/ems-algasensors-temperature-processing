
# Temperature Processing Microservice
### Alga Sensors

---

This microservice is responsible for managing the devices in the Alga Sensors system. It provides an API for creating, updating, deleting, and retrieving device information. The microservice also handles device registration and authentication.

## Features 
- Device registration
- Device authentication
- Device information management
  - Device status monitoring
    - Online/offline       status    
    - Battery level
    - Signal strength
    - Location tracking
    - Firmware version
    - Device type
    - Device model
      - 
    - Device manufacturer
    - Device serial number
      - Device owner
